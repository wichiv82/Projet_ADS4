Le systeme RGB utilise en informatique distingue 3 couleurs primaires :
  1. rouge code en hexadecimal FF0000 ;
  2. vert code 00FF00 ;
  3. bleu code 0000FF ;
Toute autre couleur etant codee par une combinaison de ces trois couleurs.
